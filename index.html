Midi to Arduino Tone

by @TawsifTorabi
Convert a MIDI file to Arduino tone code.


This is a WebApp built on javascript to convert MiDi files to Arduino Codes to make arduino play music.
Arduinos are capable of generating square wave tones which makes them able to play some tone to spice things up.
